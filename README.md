📁 Project Structure & Usage
•	main.py: The primary execution script. Running this file will immediately generate the final submission.csv.
•	cache/: This directory stores pre-calculated GLM Quality features. It was added to optimize performance and save significant execution time by avoiding recalculations during every run.
•	data/: Contains the 36 extracted CSV files provided by Kaggle for the competition.
•	previous_versions/: An archive of the project's evolution, containing all iterations from the initial version (E001) to the final submission (E028).
•	submission.csv: The final output required for the competition. To receive a leaderboard score, this file must be uploaded to the Kaggle Submission Page.
